如何写好一个PHP的类
======

## 课程介绍

首先，这门课面向PHP开发者，主要讲解一个类库的的建立、开发和使用。课程内容会逐步涉及到PHP在面向对象开发过程中使用到的常用知识。
与理论中强调大而全不同的是，这里面讲到的都是偏实用的东西。这里面会涉及到一些经常听说到但对于很多人来说很少用到的东西，如：

1. 测试驱动开发（TDD）以及测试基类的原理
2. 如何使用配置
3. 多级子类的使用
4. 使用异常
5. 常用的类的写法和优化
6. 重构
7. spl里的一些东西
8. 敏捷的一些原则和实践
9. 除此之外的其它知识

于此同时，我个人也在不断学习中，有好的东西也会在这里和大家分享。
